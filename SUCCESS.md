# Global Traders - Premium Corporate Website

The Global Traders website has been designed as a high-fidelity, futuristic enterprise showcase. It combines deep luxury aesthetics with cutting-edge animations to create an immersive global logistics and procurement experience.

## Key Features

- **Dark Luxury Interface**: A sophisticated palette of matte black and deep navy, accented by electric blue neon glows.
- **3D Interactive Globe**: A rotating wireframe globe in the hero section, representing the company's global reach and interconnected trade routes.
- **Cinematic Animations**: Smooth GSAP-powered transitions and scroll-reveal effects that bring the corporate story to life.
- **Pinterest-Inspired Masonry**: An elegant, staggered grid for core services, providing a modern and premium feel.
- **Horizontal Project Timeline**: A clear, animated breakdown of the project execution process, from consultation to delivery.

## Visual Showcase

### Hero Section
The hero section features a dramatic fullscreen landing page with a rotating 3D globe and bold animated typography.

### About Us & Core Services
Premium glass-morphism cards and hover-glow effects highlight the company's expertise and global network.

### Project Execution Timeline
A streamlined horizontal timeline that visualizes the end-to-end delivery process with glowing pulse animations.

**Note**: The website is fully responsive and optimized for both desktop and mobile viewing, maintaining its premium aesthetic across all devices.
